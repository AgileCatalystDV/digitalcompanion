# Security Audit (Mobile/BLE)

## Description

Scope: BLE pairing/encryption/MITM; WiFi P2P isolation; OWASP Mobile Top 10; permissions.

## Instructions

1. Identify scope: BLE, WiFi, permissions, API calls
2. Apply OWASP Mobile Top 10
3. Check BLE: pairing, encryption, MITM
4. Check WiFi P2P: network isolation
5. Output: Executive Summary, Findings (Critical/High/Medium/Low), Recommendations

## Output Template

```markdown
# Security Audit Report
## Executive Summary
## Findings (Critical/High/Medium/Low)
## Recommendations
## Remediation Checklist
```
